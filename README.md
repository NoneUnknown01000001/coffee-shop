# ☕ Coffee Shop Website

A responsive and modern coffee shop website designed to showcase menu items, services, and create an engaging user experience.  
This project focuses on clean design, smooth animations, and mobile-friendly layouts.

---

## 🚀 Features
- Fully responsive design for desktop, tablet, and mobile
- Built with **React**, **Tailwind CSS**, and **Bootstrap**
- Smooth animations powered by **GSAP**
- Sections for menu, about, services, and contact
- Easy to customize for any café or restaurant
